七彩LED投影灯

使用ESP8266开发的物联网变色LED灯

还具备Android及PC客户端,可通过有线/无线方式更改灯的颜色及动画等属性

MCU:

ESP-12F 4MB Flash 80KB RAM

外围设备:

ch340

WS2801的LED灯板

外壳:

3D打印 模型位于项目根目录下

第三方库:

ESP8266 Arduino Core

FastLED

ArduinoJson

arduinoWebSockets

PS: 如果在配置VSCode开发环境时遇见头文件丢失的问题可参考https://github.com/microsoft/vscode-arduino/issues/438(然而并不好用)
